项目组合：
1.face_image文件夹：用来存放员工人脸图片
2.page文件夹：主要的逻辑代码都在这个文件夹内
    2.1 homePage.py：主页，主要包括人脸考勤打卡、新用户注册跳转、管理界面跳转等功能
    2.2 enrollPage.py：注册页面，主要包括注册功能, 拍照注册、选择图片注册，填写用户的基本信息
    2.3 managePage.py：管理页面，主要包括管理员注册与登录、员工考勤打卡、请假管理、考勤统计等功能
3.sql文件夹：主要存放数据库相关代码
4.ui文件夹：主要存放UI相关代码, 这个代码你不需要去看，只是画界面的，没有逻辑
5.faceRecognition.py：(测试用)人脸识别相关代码，主要包括人脸检测、特征提取、人脸比对等功能
6.run.py：运行文件，主要是启动项目
